# Bruno 🇹🇿

**Bruno** is an open-source AI companion built and developed in Tanzania. He knows everything about Tanzania history, culture, geography, people, economy, wildlife, daily life, and more.

---

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- Google Gemini API
- Supabase Edge Functions

---

## Features
✅ General knowledge about all things Tanzania
✅ Responsive design for mobile & desktop
✅ Multi-language support
✅ Open source & easy to customize
